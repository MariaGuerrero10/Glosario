GLOSARIO DE TI

Inicio N O P Q R S T U V W X Y Z

A - M

N

Nube: Conjunto de servidores conectados en red que se utilizan para almacenar y procesar datos.

O

Objetivo: Meta o finalidad que se desea lograr en un proyecto o proceso.

P

Proceso: Serie de actividades o pasos que se llevan a cabo para lograr un objetivo o resultado.

Q

QA (Calidad): Proceso de aseguramiento de la calidad que se lleva a cabo durante el desarrollo de software para garantizar que el producto final cumpla con los requisitos y estándares establecidos.

R

Reto: Dificultad o problema que se enfrenta en un proyecto o proceso.

S

Software: Conjunto de instrucciones y datos que se ejecutan en un dispositivo electrónico para realizar tareas específicas.

T

Tecnología: Conjunto de conocimientos y habilidades que permiten el desarrollo y aplicación de herramientas y métodos para solucionar problemas o mejorar procesos.

U

UML (Unified Modeling Language): Lenguaje de modelado unificado que se utiliza para representar y documentar el diseño y la estructura de un sistema de software.

V

Valoración: Proceso de evaluar y asignar un valor a un producto o servicio en función de sus características y beneficios.

W

Web: Red de información global que permite a los usuarios acceder a información a través de dispositivos electrónicos como computadoras, teléfonos inteligentes y tabletas.

X

XHTML (Extensible Hypertext Markup Language): Versión de HTML que sigue las reglas de XML y se utiliza para crear documentos web que son más fáciles de leer y procesar por los navegadores.

Y

Youtuber: Persona que crea y comparte videos en línea en plataformas como YouTube, generalmente con fines de entretenimiento y monetización.

Z

Zona de transferencia: Área geográfica donde se pueden encontrar productos o servicios que se han importado de otros países.
