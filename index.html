A - M

A

API (Interfaz de Programación de Aplicaciones): Un conjunto de reglas que permite que dos aplicaciones se comuniquen entre sí.

B

Big Data: Conjuntos de datos extremadamente grandes que requieren herramientas especiales para su procesamiento y análisis.

C

Ciberseguridad: Prácticas y tecnologías utilizadas para proteger sistemas, redes y datos de amenazas y ataques cibernéticos.

D

Data Center: Un centro de procesamiento de datos que almacena, gestiona y distribuye información.

E

Email (Correo Electrónico): Sistema de comunicación que permite enviar y recibir mensajes a través de internet.

F

Firewall: Software o hardware que protege una red contra accesos no autorizados.

G

Gigabit: Unidad de medida de velocidad de transferencia de datos, equivalente a 1.000 millones de bits por segundo.

H

Hardware: Componentes físicos de un sistema informático, como procesadores, memorias, discos duros, tarjetas gráficas, etc.

I

IP (Dirección IP): Dirección única que identifica a un dispositivo en una red.

J

Jabber: Sistema de mensajería instantánea que permite enviar y recibir mensajes a través de internet.

K

Kilobyte: Unidad de medida de capacidad de almacenamiento, equivalente a 1.024 bytes.

L

Laptop: Ordenador portátil, que permite llevar a cabo tareas informáticas en cualquier lugar.

M

Máquina virtual: Programa que simula un entorno de hardware y software, permitiendo ejecutar aplicaciones en un sistema operativo diferente al del equipo físico.
